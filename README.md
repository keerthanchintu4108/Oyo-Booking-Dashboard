# 🏨 OYO Booking Dashboard – Power BI

## 📌 Project Overview

The **OYO Booking Dashboard** is an interactive Power BI project created to analyze hotel booking data and visualize business performance. The dashboard provides insights into booking trends, revenue generation, room occupancy, and customer behavior using clear charts and key performance indicators (KPIs).

This project demonstrates data analysis and dashboard development skills using **Power BI**, helping users easily understand booking patterns and hotel performance.

---

## 📊 Features

* Interactive Power BI dashboard
* Booking trend analysis
* Revenue and occupancy insights
* Popular locations and booking patterns
* Dynamic filters and slicers
* Clear charts and KPI indicators

---

## 📂 Project Files

* **OYO Data.xlsx** – Dataset used for analysis
* **Oyo Booking dashboard.pbix** – Power BI dashboard file
* **oyo.jpeg** – Dashboard preview image

---

## 🛠 Tools & Technologies

* Power BI
* Microsoft Excel
* Data Visualization
* Data Analysis

---

## 🚀 How to Use

1. Download the repository files.
2. Open **Oyo Booking dashboard.pbix** using **Power BI Desktop**.
3. Explore the dashboard using filters and visualizations.
